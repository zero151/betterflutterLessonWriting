git init - Инициализируй локальный репозиторий(Создаёт скрытую папку .git внутри твоей папки. Она хранит всю историю изменений, настройки и метаданные. Без неё Git не работает.)
git add . - Добавь все файлы к коммиту(Добавляет все файлы из текущей папки в "область подготовки" (staging area). Git теперь знает, что эти файлы нужно включить в следующий коммит.)
git commit -m "Первый коммит" - Сделай коммит(Фиксирует изменения как "снимок" (snapshot) проекта. Это как сохранение в видеоигре.)
git remote add origin https://github.com/твой-юзернейм/имя-репозитория.git - Привяжи к удалённому репозиторию(Говорит Git'у: "Этот локальный репозиторий связан с удалённым по адресу...".
origin — это стандартное имя для основного удалённого репозитория.)
git branch -M main
git push -u origin main  - Загрузи файлы на GitHub
